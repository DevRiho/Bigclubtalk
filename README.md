# Big Club Talk

Big Club Talk is a white-first sports publishing platform for football news, transfers, analysis, club stories, player features, fan opinion, and breaking news.

## Apps

- `frontend`: React, Vite, Tailwind CSS, React Router, TanStack Query, Axios, Framer Motion, React Hook Form.
- `backend`: Node.js, Express, MongoDB, Mongoose, JWT auth, refresh tokens, Cloudinary, Nodemailer, Express Validator.

## Local Setup

1. Copy `frontend/.env.example` to `frontend/.env`.
2. Copy `backend/.env.example` to `backend/.env`.
3. Run `npm install`.
4. Run `npm run dev`.

The frontend defaults to `http://localhost:5173`; the backend defaults to `http://localhost:5000`.
